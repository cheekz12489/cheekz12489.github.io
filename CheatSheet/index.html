HTML & CSS Reference


Languages For Web Development

Language Description
HTML Hypertext Markup Language - Structures web content
CSS Cascading Style Sheets - Applies styling to web content
JavaScript Adds interactivity to web content
SQL Structured Query Language - Allows storing & retrieval of data


HTML Tags

Tag Name Description
<!DOCTYPE html> Document Type Self-Closing tag to indicate what version of html will be used.
<!--- TEXT ---> Comment Self-Closing tag used to insert comments into HTML file - No visual effect on page.
<html> HTML Container to indicate content that will be interpreted as HTML.
<head> Head Wrapper element for metadata of the web page - Content will not be displayed.
<style> Style Container used to allow CSS rulesets to be placed with HTML documnet itself.
<title> Title Wrapper element for Title of the webpage - Displayed in tab of browser.
<meta> Metadata Container used to provide information about the webpage placed inside the Head Section - Often used to indicate the page discription, character set, author, or viewpoint settings.
<body> Body Wrapper element for content that will be displayed to the screen
<div> Division Divides page into sections - No visual effects.
<h1> Heading 1 Largest heading.
<h2> Heading 2 Second largest heading.
<h3> Heading 3 Third largest heading.
<h4> Heading 4 Third smallest heading.
<h5> Heading 5 Second smallest heading.
<h6> Heading 6 Smallest heading.
<p> Paragraph Content within tags will be formatted like a paragraph.
<span> Span Inline container to group small pieces of text or other elements - No visual change.
<em> Emphasis Inline container to italicize text.
<strong> Highlight Inline container to make text bold.
<br> Break Self-Closing tag to insert a line break within the browser.
<hr> Horizontal Line Self-Closing tag to insert a horizontal line in the webpage.
<ul> Unordered List Container for bulleted list items.
<ol> Ordered List Container for numbered list items.
<li> List Items Container for items placed within an ordered or unordered list.
<img> Image Self-closing tag to display an image in the browser.
<a> Anchor Container allowing text within to link to another document or element on the page.
<table> Table Wrapper element for HTML tables.
<tr> Table Row Conyainer to insert a row into a table.
<td> Table Data Container to add cell content to a table row.
<th> Table Header Container to add table headers to a column within a row.
<tbody> Table Body Container to help section off large tables - Includes everything except the headings.
<thead> Table Head Container to section off rows containing column headers in a table.
<tfoot> Table Foot Container to section off the foot of a table - Usually contains totals, results, or other summary data.
<header> Header Semantic container used for intrudcutory information - Often contains navigation links or headings.
<nav> Navigation Semantic container often used for navigation links or table of contents.
<main> Main Semantic container used to section off dominant content of a webpage.
<footer> Footer Semantic container placed at the bottom of the mage - Often used for contact/copyright/terms of use/site map info or links to the top of the page.
<section> Section Semantic container to group areas of a page with the same theme/topic - Often used for things like chapters or headings.
<article> Article Semantic container used for conten that makes sense on its own - Often used for things like blogs, articles, commnets, or magazines.
<aside> Aside Semantic container used to include additional info that enhances another element, but that is not required - Often used for things like bibliographies, comments, end notes, etc.
<figure> Figure Semantic container used to encapsulate media referenced by the main content - Often used for things like images, illustrations, diagrams, or code snippets.
<figcaption> Figure Caption Semantic container used to describe the contents of the Figure Tag.
<video> Video Container to display a video - Text within tag only displayed if video cannot be loaded.
<audio> Audio Semantic container used to embed audio content into the webpage.
<source> Source Semantic container used to identify the source of the audio element.
<track> Track Self-Closing tag used inside the Audio or Video Tag to specify tracks, such as subtitles, captions, or descriptions about the media content.
<embed> Embed Semantic container (Self-Closing Tag) used to embed any medio content to the webpage - Often used for video, audio, or gifs. **Deprecated**
<canvas> Canvas Container used to place a space on the page where you can draw graphics using JavaScript.
<svg> Scalable Vector Graphics Container used to draw shapes, lines, and text using XML markup - SVG images scale without losing quality and are good for icons, charts, and illustrations.
<script> Script Container used to includ JavaScript code within an HTML documnet - Usually placed within the <head> Tag or at the end of the <body> Tag.

HTML Attributes

Attribute Name Description Values
href Hyperlink Reference Identifies the destination location of the Hyperlink for <a> Tags. Absolute/Relative Path or Jump to #UniqueID
rel Relationship Describes the relationship between the current document and the linked resource. stylesheet, noopener, noreferrer, nofollow, alternate, icon, author
id ID Uniquely identifies an element by a user-defined name. User Defined
class Class Assigns one or more user-defined names that can be used for styling multiple elements at once. User defined
src Source Specifies the source file for images, audio, video, etc. Absolute/Relative Path
alt Alternative Text Provides alternative text for images if image cannot load and for assistive screen readers. User Defined
width Width Sets the size for images, video, or other elements that can be sized. px, %, em, rem, vw, vh
height Height Sets the size for images, video, or other elements that can be sized. px, %, em, rem, vw, vh
controls Controls Includes basic video or audio controls for embedded media. None
target Target Specifies where to open the linked document _blank, _self, _parent, _top
colspan Column Span Defines how many columns a cell should span horizontally within a table row. Integer Value
rowspan Row Span Defines how many rows a cell should span verically with a table column. Integer Value
scope Scope Specifies what part of a table the table header refers to, such as a column or row. col, row, colgroup, rowgroup
type Type Defines the type or format of an element, such as input, button, script, or link. <input> - text, password, email, numnber, checkbox, radio, submit
<button> - button, submit, reset
<script> - text/javascript, module
<link> - text/css, image/png
<ol> - 1, A, a, I, i
autoplay Auto Play Used with audio and video tags to force media start playing as soon as the webpage is loaded. None
loop Loop Used with audio and video tags to force media to play again automatically once it has ended. None
title Title Adds extra information shown as a tooltip. User Defined

CSS Properties

Property Name Description Value
min-width Minimum Width Sets minimum width an element can be - Used to prevent content from shrinking too much when screen size gets too small. px, %, em, rem, vw, vh
max-width Maximum Width Sets maximum width an element can be, even when screen size increases past this size. px, %, em, rem, vw, vh
min-height Minumum Height Sets the minimum height an element can be, even when screen size decreases less than this size. px, %, em, rem, vw, vh
max-height Maximum Height Sets the maximum height an element can be, even when screen size increases past this size. px, %, em, rem, vw, vh
color Text Color Used to change the foreground color of a text element. Named Colors, Hex Codes, rgb()/rgba(), hsl()/hsla()
background-color Background Text Color Used to change the background color of an element. Named Colors, Hex Codes, rgb()/rgba(), hsl()/hsla()
font-family Font Family Chooses the font of the text within the element. Font Names, Generic Families, Fallback Fonts, Free Hosted, Downloaded
font-weight Font Weight Sets how bold or light the text appears. Numeric Value bt 100s, normal, bold, bolder, lighter
font-size Font Size Sets the size of the text within the element. px, pt, cm, mm, in, %, em, rem, vw, vh, xx-small, x-small, small, medium large, x-large, xx-large, smaller, larger
text-align Text Alignment Sets the horizontal alighment of the text within the element. left, right, center, justify
opacity Opacity Sets the transparency of an element. Decimal between 0(Invisible) and 1(Fully Visible)
background-image Background Image Sets an image as the background of an element. url(Absolute/Relative Path)
list-style List Style Sets the style of the list item markers, such as the bullets or the numbers. disc, circle, square, decimal, lower-alpha, upper-alpha, none
padding Element Padding Sets the space between the content and the border of an element. px, %, em, rem (1-4 Values)
padding-top Top Padding Sets the space between the top of the content and the border of an element px, %, em, rem
padding-right Right Padding Sets the space between the right side of the content and the border of an element. px, %, em, rem
padding-bottom Bottom Padding Sets the space between the bottom of the content and the border of an element. px, %, em, rem
padding-left Left Padding Sets the pace between the left side of the content and the border of an element. px, %, em, rem
border-radius Border Radius Rounds the corners of an element's border. px or % (1 to 4 Values)
cursor Cursor Changes the mouse pointer when hovering over an element. pointer, default, text, move, not-allowed, wait
border Border Sets an element's border width, style, and color. px, em (Width), solid, dashed, dotted, double, groove (Style), Any Color Variation
margin Margin Sets the space outside of the border between neighboring elements. px, %, em, rem (1 to 4 Values)
margin-top Top Margin Sets the space outside of the top of an element's border and the element above it. px, %, em, rem
margin-right Right Margin Sets the space outside of the right side of an element's border and the element to its right. px, %, em, rem
margin-bottom Bottom Margin Sets the space outside of the bottom of an element's border and the element below it. px, %, em, rem
margin-left Left Margin Sets the space outside of the left side of an element's border and the element to the left. px, %, em, rem
overflow Overflow Controls what happens if content overflows an element's box. visible, hidden, scroll, auto
overflow-x X Overflow Handles overflow on the left and right sides of an element. visible, hidden, scroll, auto
overflow-y Y Overflow Handles overflow on the top and bottom of an element. visible, hidden, scroll, auto
visibility Visibility Shows or hides and element without completely removing it from the webpage flow. visible, hidden, collapse
display Display Sets how an element is displayed within the webpage. block, inline, inline-block, none, flex, grid
box-sizing Box Sizing Controls how width and height are calculated when considering things like padding and margin. content-box, border-box
position Position Sets how an element is positioned in terms of the flow of other elements within the webpage. static, relative, absolute, fixed, sticky
z-index Z Index Sets the stacking order of positioned elements - Determines the depth of an element. Any integer, negative elements can be behind others, auto follows HTML order
float Float Moves an element to the left or right, allowing text and inline elements to wrap around it. left, right, none, inherit
clear Clear Controls the behavior of floating elements that bump into each other. left, right, both, none, inherit
font-style Font Style Sets the style of the font. normal, italic, oblique, inherit
text-transform Text Transform Changes the case of the text within the element. none, uppercase, lowercase, capitalize, inherit
letter-spacing Letter Spacing Sets the space between letters within the element. px, em, rem, normal
word-spacing Word Spacing Sets the space between words within the element. px, em, rem, normal
line-height Line Height Sets the vertical space between lines of text. px, %, em, rem, normal
flex-direction Flex Direction Specifies which direction the flex elements will be displayed. row, row-reverse, column, column-reverse
align-items Aliign Items Aligns flex items within the flex container. center, baseline, stretch, flex-start, flex-end
justify-content Justify Content Used to align flex and grid items within their container. flex-start, flex-end, center, space-between, space-around, space-evenly
transition Transition Change property values over a set time duration. property, duration, timing function(ease, linear), delay
box-shadow Box Shadow Adds shadow effect around element's frame. horizontal offset, vertical offset, blur radius, spread radius, color
text-shadow Text Shadow Adds shadow to text. horizontal offset, vertical offset, blur radius, color
